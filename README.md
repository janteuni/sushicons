# Sushicons


## Install
get bower package

    bower install sushicons --save

link dependencies

```css
<link rel="stylesheet" href="bower_components/sushicons/dist/style.css">
```

## Usage
add class on your html element

    <i class="si-lasvegas"></i

and style icon

```css
i {
  font-size: 21px;
  color: #37474F;
}
```
